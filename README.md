#### 2. 애프리케이션 실행 방법 with Docker

2-1. git repository clone

```bash
git clone https://github.com/MinChangJeong/wanted-pre-onboarding-backend.git
```

2-2. gradle build

```bash
./gradlew build
```

2-3. docker compose up(Daemon)

```bash
docker compose up --build -d
```
